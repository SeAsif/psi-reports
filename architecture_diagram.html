Data Flow Diagram — One-Way Pipeline

The data pipeline is strictly ONE-WAY. SQL Server is the untouchable source of truth. Data flows forward into BigQuery (the cloud warehouse), and reports flow out to Google Sheets. Nothing ever writes back to SQL Server. The app has no SQL Server connection at all.
